# 博客模板 v1.0-template

## 版本信息

- 发布版本：v1.0
- 主题版本：[vdoing](https://github.com/xugaoyi/vuepress-theme-vdoing) - v1.12.7
- 发布时间：2022-08-05

## 内容

- 博客页面：首页、索引页、文章页、分类、标签、归档、友链、收藏
- 自动化部署：服务器、仓库gh-pages分支、百度推送
- Waline、Gitalk评论系统
- 博客数据统计显示
- 文章数据统计显示
- rss、搜索插件、sitemap
- ...
